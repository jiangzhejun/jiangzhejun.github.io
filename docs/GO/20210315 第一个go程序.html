2021/3/15 北京

一、下载并安装

  1. 前往官网，下载免安装压缩包go1.16.2.windows-amd64.zip，解压到安装目录。
  2. 添加环境目录 set path=D:\go\bin;%path%
  3. 验证安装成功

E:\bibi\junet>go version
go version go1.16.2 windows/amd64

二、编写源代码

package main

import "fmt"

func main() {
  fmt.Println("Let's go!")
}

三、测试运行

E:\bibi\junet\lab>go run let's.go
Let's go!

四、编译可执行文件

E:\bibi\junet\lab>go build let's.go

E:\bibi\junet\lab>let's
Let's go!
