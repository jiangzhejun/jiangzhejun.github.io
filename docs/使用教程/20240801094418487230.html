SCP命令详解

2024.8.1 北京

SCP全称是Secure Copy Protocol，它是一个命令行工具，允许用户在两个主机之间安全的复制文件和目录。SCP通过SSH连接加密传输，确保数据传输过程中不会被攻击。简单、安全，在大多数Unix系统下都是预安装的，Windows 10以上系统也已内置。

SCP默认加密算法是AES-128。复制相同文件时，会默认进行覆盖，没有任何的警告提示。复制大文件时，终端可能会睡眠终止传输过程，建议使用tmux来维持会话。

scp 参数表 源文件列表 用户名@目标主机地址:文件路径


不传值

  • -1 强制使用SSH1协议
  • -2 强制使用SSH2协议
  • -3 两台远程主机之间的副本通过本地主机传输
  • -4 使用IPv4
  • -6 使用IPv6
  • -A 允许将ssh代理转发到远程系统
  • -B 批处理模式（阻止终端询问ssh密码）
  • -C 开启压缩传输
  • -p 保留文件的修改、访问时间和权限模式
  • -q 不显示输出、调试信息
  • -r 以递归的方式进行复制
  • -T 禁用严格的文件名检查
  • -v 显示输出、调试信息


传值

  • -c 指定传输数据的加密算法
  • -F 指定SSH配置文件
  • -i 指定读取用于公钥身份验证的标识（私钥）的文件
  • -J 指定1个以上的跳转主机（逗号分隔）
  • -l 限制使用的带宽
  • -o 指定使用的ssh选项
  • -P 指定目标主机的端口
  • -S 用于加密连接的程序名称
