Debian10安装配置TinyProxy

姜哲俊 2021年11月16日

1. 安装

apt-get update apt-get install tinyproxy

2. 配置

vim /etc/tinyproxy/tinyproxy.conf Port 8888 #Allow 127.0.0.1

3. 启动

service tinyproxy start

4. 设置防火墙规则

iptables -I INPUT -p tcp --dport 8888 -j ACCEPT iptables -L

5. 保存防火墙规则

apt-get install iptables-persistent netfilter-persistent save netfilter-persistent reload
