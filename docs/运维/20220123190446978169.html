JDK15 & Tomcat9 for CentOS 7.8



JDK



安装
  • 官网下载 openjdk-15.0.1_linux-x64_bin.tar.gz 到 /bww/res
  • tar -zxvf openjdk-15.0.1_linux-x64_bin.tar.gz
    解压缩
  • mv jdk-15.0.1 /bww/apps/jdk-15.0.1
    移至安装目录


设置环境变量
  • vi /etc/profile
    打开配置文件
  • 键入
    i
    切换输入模式
  • 光标移至最后一行输入：
    • export JAVA_HOME=/bww/apps/jdk-15.0.1
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
export PATH=$JAVA_HOME/bin:$PATH
  • 按 ESC 退出输入模式
  • 键入
    :wq
    存盘退出
  • reboot
    重启系统
  • java -version
    查看是否安装成功


Tomcat



安装
  • 官网下载 apache-tomcat-9.0.39.tar.gz 到 /bww/res
  • tar -zxvf apache-tomcat-9.0.39.tar.gz
    解压缩
  • mv apache-tomcat-9.0.39 /bww/apps/apache-tomcat-9.0.39
    移至安装目录
  • vi /bww/apps/apache-tomcat-9.0.39/conf/server.xml
    修改默认端口
    • <Connector port="80" protocol="HTTP/1.1" … />
  • /bww/apps/apache-tomcat-9.0.39/bin/startup.sh
    启动服务


设置防火墙
  • systemctl start firewalld
    启动防火墙
  • systemctl status firewalld
    查看防火墙状态
  • firewall-cmd --zone=public --add-port=80/tcp --permanent
    开启端口
  • firewall-cmd --reload
    重启防火墙
  • firewall-cmd --query-port=80/tcp
    查看端口是否开启
