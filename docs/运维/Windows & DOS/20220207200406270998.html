Windows家庭版开启组策略

2021/3/5 北京

新建文本文件，键入一下代码： 
@echo off & title %~n0 - bi201907041144
set dona=dism /online /norestart /add-package:
set cwsp=%systemroot%\servicing\Packages
pushd "%~dp0"

dir /b %cwsp%\Microsoft-Windows-GroupPolicy-ClientExtensions-Package~3*.mum>$
dir /b %cwsp%\Microsoft-Windows-GroupPolicy-ClientTools-Package~3*.mum>>$
for /f %%i in ('findstr /i . $ 2^>nul') do %dona%"%cwsp%\%%i"
del /f/q $
pause
保存为批处理文件，以管理员身份运行。结束后Win+R输入
gpedit.msc
启动“本地组策略编辑器”，如果报错重启系统即可。
