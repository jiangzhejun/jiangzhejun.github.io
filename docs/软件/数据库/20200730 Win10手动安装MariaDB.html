Win10手动安装MariaDB

2020年7月30日

安装/启用

  1. 前往官网（https://downloads.mariadb.org/）下载安装包，如：mariadb-10.5.4-winx64.zip
  2. 解压缩到指定目录D:\MariaDB，并创建数据目录data
  3. 以管理员身份运行命令行窗口，定位到安装目录\bin
  4. 执行安装命令 mysql_install_db.exe --datadir=D:\MariaDB\data --service=MariaDB --password=123456
  5. 启停服务 net start/stop MariaDB
  6. 登录 mysql -u root -p（按提示输入密码）

修改编码

  1. MariaDB [(none)]> set character_set_client=utf8;
  2. MariaDB [(none)]> set character_set_connection=utf8;
  3. MariaDB [(none)]> set character_set_database=utf8;
  4. MariaDB [(none)]> set character_set_results=utf8;
  5. MariaDB [(none)]> set character_set_server=utf8;
  6. MariaDB [(none)]> set character_set_system=utf8;
  7. MariaDB [(none)]> show variables like 'character%';
