Tomcat9 & JDK11 安装配置手记

2018-10-29 17:40

JDK

官网下载openjdk-11.0.1_windows-x64_bin.zip并解压缩至D盘

Tomcat

  • 官网下载apache-tomcat-9.0.12-windows-x64.zip并解压缩至D盘
  • 修改service批处理文件
    • 进入安装目录bin文件夹
    • 记事本打开service.bat文件
    • 插入“set JAVA_HOME=D:\jdk-11.0.1\”和“set JRE_HOME=%JAVA_HOME%”两行
  • 修改server配置
    • 进入安装目录conf文件夹
    • 记事本打开server.xml文件
    • 找到“<Connector port=“8080” protocol=“HTTP/1.1” … />”，修改端口为80
    • 滚动条拉到底，找到默认主机“<Host name=“localhost” …>”，插入“<Context path=”” debug=“0” docBase=“D:/website” … />”，自定义根目录
  • 必要的类库
    • 进入网站根目录
    • 将编译好的class文件拷贝至classes文件夹，没有就新建一个
    • 将编译好的jar文件拷贝至lib文件夹，没有就新建一个
  • 安装Tomcat9服务
    • 打开CMD
    • 切换至安装目录bin文件夹
    • 运行“service install Tomcat9”
    • 运行“services.msc”打开系统服务
    • 启动服务
